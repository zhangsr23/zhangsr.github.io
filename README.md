## difference between algorithm and model
- 算法：是一系列清晰、明确、可执行的指令或步骤的集合，用于解决特定的问题或完成特定的任务，侧重于描述解决问题的具体操作过程和逻辑。
- 模型：对事物、现象的抽象、简化的表示和描述，通常用一组数学公式、规则、结构来模拟、解释、预测事物或现象的发展走向。
